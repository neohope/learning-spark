About
====================
This is a code example that complements the material in the Spark O'Reilly book. 


Requirements
====================
1.JDK1.8+

2.Maven

3.Scala相关的Jar包，建议用2.11.x版本，2.12.x有些Jar不支持


Build
====================
```shell
mvn package
```


Run
====================
1. use IDE like eclipse to run the demos
